# Loadhunt Scroll Landing

Vite + React landing page for Loadhunt / LoadHunter.

## Vercel Settings

Use:

```text
Framework Preset: Vite
Root Directory: /
Install Command: npm install --legacy-peer-deps --no-audit --no-fund --no-package-lock
Build Command: npm run build
Output Directory: dist
Node.js Version: 22.x
```

Do not set Root Directory to `loadhunt-scroll-landing`.

## Local Run

```bash
npm install --legacy-peer-deps --no-audit --no-fund --no-package-lock
npm run dev
```

## Build

```bash
npm run build
```
